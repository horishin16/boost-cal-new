# 要件定義書 v2: 予定調整作成画面（P003）

> DB設計を踏まえた要件定義書の更新

## 目的

スケジュールリンクを新規作成または編集する。参加者の選択、対応可能時間帯の設定、会議オプションの設定を行い、共有用URLを生成する。

## 機能

| # | 機能 | 説明 | DB関連テーブル |
|---|------|------|--------------|
| 1 | タイトル・説明入力 | リンクの名前と説明を入力 | schedule_links (name, description) |
| 1b | カスタムスラッグ設定 | リンクURLの末尾を任意の文字列に設定・編集 | schedule_links (slug: UNIQUE制約チェック) |
| 2 | 参加者選択 | 社内メンバー検索・選択、外部メール追加。連携済み外部ユーザーは空き時間計算に含む | users (READ: 検索), schedule_links.settings (participants) |
| 2b | カレンダー連携招待 | 外部参加者にカレンダー連携招待リンクを送信 | calendar_invitations (CREATE), users (READ: GUEST確認) |
| 3 | 会議時間選択 | 会議の所要時間を選択（15/30/60/90分） | schedule_links.settings (allowedDurations) |
| 4 | 対応可能時間帯設定 | カレンダーグリッドで曜日別の時間帯をドラッグ設定 | schedule_links.settings (weekdayTimeSlots) |
| 4b | 祝日除外設定 | 祝日を自動的に調整不可にするON/OFF | schedule_links.settings (excludeHolidays) |
| 4c | 日付オーバーライド | 特定日に個別の調整枠を設定 | schedule_links.settings (dateOverrides) |
| 5 | 空き時間自動計算 | 参加者のカレンダーから空き時間を自動抽出・表示 | users (READ: access_token, refresh_token) + Google Calendar API |
| 6 | 会議形式選択 | 許可する形式を選択（複数選択可） | schedule_links.settings (meetingOptions) |
| 7 | バッファー時間設定 | 形式ごとのバッファー時間を設定 | schedule_links.settings (meetingOptions.buffer*) |
| 8 | 会議室選択 | 対面（社内）許可時に会議室を選択 | schedule_links.settings (conferenceRoomId) + Google Workspace API |
| 9 | 訪問先入力 | 対面（訪問）許可時に先方の場所名・住所を入力 | schedule_links.settings (visitLocation) |
| 10 | リンク生成・コピー | 保存してURLを生成・コピー | schedule_links (CREATE/UPDATE) |
| 11 | 既存設定ロード | 編集モード時に既存設定を読み込み | schedule_links (READ: id) |

## Input / Process / Output

| 機能 | Input | Process | Output | DB関連テーブル |
|------|-------|---------|--------|--------------|
| タイトル・説明入力 | タイトル（必須、最大100文字）、説明（任意、最大500文字） | バリデーション（タイトル必須・文字数チェック） | フォームに値が反映される | schedule_links.name, .description |
| カスタムスラッグ設定 | スラッグ文字列（任意、最大50文字、半角英数字・ハイフンのみ） | 未入力時は自動生成。入力時はユニーク制約チェック（API問い合わせ） | URLプレビュー表示。重複時はエラー | schedule_links.slug (UNIQUE制約) |
| 参加者選択 | 検索キーワード or 外部メールアドレス | 社内: APIで組織メンバー検索。外部: GUEST登録済みか確認 | 選択済みリストに追加（連携ステータス表示） | users (READ: email, name, role), schedule_links.settings.participants |
| カレンダー連携招待 | 未連携外部参加者の「連携を依頼」ボタンクリック | 招待リンク生成・メール送信 | メール送信完了。連携完了後ステータス更新 | calendar_invitations (CREATE: token, email, inviter_id), users (連携後 CREATE as GUEST) |
| 会議時間選択 | 会議時間ボタン（15/30/60/90分） | 選択した時間を設定に反映 | ハイライト表示 | schedule_links.settings.allowedDurations |
| 対応可能時間帯設定 | カレンダーグリッド上でのドラッグ操作 | ドラッグ範囲から時間帯を算出し曜日別デフォルトとして設定に追加 | 対応可能時間帯ブロック（青色）を表示 | schedule_links.settings.weekdayTimeSlots |
| 祝日除外設定 | トグルON/OFF | 祝日（@holiday-jp）を調整不可に | カレンダーグリッドの祝日表示が更新 | schedule_links.settings.excludeHolidays |
| 日付オーバーライド | 特定日クリック→個別スロット設定 | 曜日別デフォルトを上書き。空配列=除外 | オーバーライド枠表示 | schedule_links.settings.dateOverrides |
| 空き時間自動計算 | 参加者リスト、対応可能時間帯設定 | 社内: DWD経由。連携済み外部: 保存済みトークン。未連携: スキップ。ビジー・祝日・過去を除外 | 空き時間（青）とビジー（灰）を表示 | users.access_token, .refresh_token (READ) + Google Calendar API |
| 会議形式選択 | チェックボックス（複数選択可） | 形式に応じたUI表示切替 | チェック済みの形式が設定に保存 | schedule_links.settings.meetingOptions |
| バッファー時間設定 | 形式ごとのバッファー分数 | 設定に保存 | デフォルト値表示 | schedule_links.settings.meetingOptions.buffer* |
| 会議室選択 | ドロップダウンから選択 | Google Workspace会議室リソース一覧取得 | 選択した会議室が設定に反映 | schedule_links.settings.conferenceRoomId + Google Workspace API |
| 訪問先入力 | 場所名（必須）、住所（任意） | バリデーション（場所名必須） | 設定に反映 | schedule_links.settings.visitLocation |
| リンク生成・コピー | 保存ボタンクリック | 全設定バリデーション。APIでリンク作成/更新。スラッグ生成 | URL表示・コピー | schedule_links (INSERT or UPDATE) |
| 既存設定ロード | URLパラメータのリンクID | APIでリンク詳細取得 | 全フォームに既存設定反映 | schedule_links (READ WHERE id = param AND owner_id = current_user) |

## UI状態遷移

| 要素 | 対応機能 | 状態 | DB関連テーブル |
|------|---------|------|--------------|
| タイトル入力フィールド | タイトル・説明入力 | デフォルト / 入力中 / エラー | schedule_links.name |
| スラッグ入力フィールド | カスタムスラッグ設定 | デフォルト / 入力中 / チェック中 / 利用可能 / エラー | schedule_links.slug (UNIQUE検証) |
| 参加者検索フィールド | 参加者選択 | デフォルト / 検索中 / 候補なし | users (検索) |
| 選択済み参加者リスト | 参加者選択 | 空 / 1名以上（連携ステータス表示） | users.role (GUEST判定), schedule_links.settings.participants |
| 連携依頼ボタン | カレンダー連携招待 | 非表示 / 表示中 / 送信済み | calendar_invitations (status) |
| 会議時間ボタン | 会議時間選択 | 未選択 / 選択中 | schedule_links.settings.allowedDurations |
| カレンダーグリッド | 時間帯設定、空き時間計算 | ローディング / 表示中 / 参加者未選択 | users (トークン), schedule_links.settings |
| 祝日除外トグル | 祝日除外設定 | ON / OFF | schedule_links.settings.excludeHolidays |
| 会議形式チェックボックス | 会議形式選択 | 各形式ON/OFF | schedule_links.settings.meetingOptions |
| バッファー時間入力 | バッファー時間設定 | 表示 / 非表示 | schedule_links.settings.meetingOptions.buffer* |
| 会議室ドロップダウン | 会議室選択 | 非表示 / 表示中 / ローディング / 選択済み | schedule_links.settings.conferenceRoomId |
| 訪問先入力フィールド | 訪問先入力 | 非表示 / 表示中 / エラー | schedule_links.settings.visitLocation |
| 保存ボタン | リンク生成・コピー | デフォルト / 無効 / ローディング | schedule_links |
| URL表示エリア | リンク生成・コピー | 非表示 / 表示中 | schedule_links.slug |
