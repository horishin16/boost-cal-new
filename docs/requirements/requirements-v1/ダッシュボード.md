# 要件定義書: ダッシュボード（P002）

## 目的

ログイン後のホーム画面。作成済みのスケジュールリンク一覧を表示し、リンクの作成・編集・削除・URLコピーを行う。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | リンク一覧表示 | 自分が作成したスケジュールリンクの一覧を表示 | 新規 |
| 2 | リンク新規作成 | 予定調整作成画面へ遷移 | 新規 |
| 3 | リンクURLコピー | 共有URLをクリップボードにコピー | 新規 |
| 4 | リンク編集 | 既存リンクの編集画面へ遷移 | 新規 |
| 5 | リンク削除 | スケジュールリンクを削除（確認あり） | 新規 |
| 6 | Calendar連携ステータス | Google Calendarとの連携状態を表示 | 新規 |
| 7 | ログアウト | セッション終了、ログイン画面へ戻る | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| リンク一覧表示 | ログインユーザーのセッション | APIで自分のスケジュールリンク一覧を取得 | リンクカード一覧（名前、URL、参加者数、作成日） |
| リンク新規作成 | 新規作成ボタンクリック | 予定調整作成画面（P003）へルーティング | P003が新規作成モードで表示 |
| リンクURLコピー | URLコピーボタンクリック | スケジュールリンクの公開URLをクリップボードにコピー | トースト「URLをコピーしました」表示 |
| リンク編集 | 編集ボタンクリック（対象リンクID） | 予定調整作成画面（P003）へリンクIDと共にルーティング | P003が編集モードで表示（既存設定読み込み済み） |
| リンク削除 | 削除ボタンクリック → 確認ダイアログで「削除」選択 | APIでスケジュールリンクを削除 | リンク一覧から該当カードが消える。トースト「削除しました」表示 |
| Calendar連携ステータス | ページ読み込み時 | ユーザーのGoogleアクセストークンの有効性を確認 | 連携済み→正常表示。未連携/期限切れ→警告バナー表示 |
| ログアウト | ログアウトボタンクリック | セッションCookie削除、Supabaseセッション終了 | ログイン画面（P001）へリダイレクト |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| リンクカード一覧 | リンク一覧表示 | ローディング / 一覧表示 / 空状態（「まだリンクがありません」） |
| 新規作成ボタン | リンク新規作成 | デフォルト / 無効（Calendar未連携時） |
| URLコピーボタン | URLコピー | デフォルト / コピー完了（一時的にアイコン変化） |
| 編集ボタン | リンク編集 | デフォルト |
| 削除ボタン | リンク削除 | デフォルト / 確認ダイアログ表示中 |
| 削除確認ダイアログ | リンク削除 | 非表示 / 表示（「本当に削除しますか？」） |
| Calendar連携バナー | Calendar連携ステータス | 非表示（連携済み） / 警告表示（未連携） |
