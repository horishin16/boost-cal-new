# 要件定義書: 調整回答画面（P004）

## 目的

共有URLを受け取ったゲストが、空き時間スロットを確認して予約する。認証不要で利用可能。Google/Microsoftログイン連携（任意）で自分の予定をオーバーレイ表示できる。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | 空き時間スロット表示 | 主催者の空き時間をカレンダービューで表示 | 新規 |
| 2 | 日付選択 | カレンダーから予約したい日付を選択 | 新規 |
| 3 | 時間スロット選択 | 利用可能な時間枠を選択 | 新規 |
| 4 | 会議時間選択 | 許可された会議時間から選択 | 新規 |
| 5 | カレンダー連携（任意） | Google/MSログインで自分の予定をオーバーレイ表示 | 新規 |
| 6 | 会議モード選択 | オンライン/対面を選択 | 新規 |
| 7 | 会議ツール指定（任意） | Zoom/Teams等のカスタムURLを入力 | 新規 |
| 8 | 予約情報入力 | 名前・メール・メモを入力 | 新規 |
| 9 | 予約確定 | 予約を確定し全員にカレンダー招待を送信 | 新規 |
| 10 | 予約完了表示 | 予約完了の確認情報を表示 | 新規 |
| 11 | エラー表示 | 無効なリンクへのアクセス時にエラー表示 | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| 空き時間スロット表示 | URLのスラッグ、選択中の会議形式 | APIでスケジュールリンク情報を取得。選択中の形式に対応するバッファー時間を加味して空き時間スロットを計算 | カレンダービューに空き日付をハイライト表示。リンク情報（主催者名、タイトル、説明）をヘッダーに表示 |
| 日付選択 | カレンダー上の日付クリック | 選択した日付の利用可能スロットをフィルタ | 右側にその日の時間枠リストを表示 |
| 時間スロット選択 | 時間枠リストからクリック | 選択したスロットを予約候補として保持 | 選択中のスロットがハイライト。予約フォームが表示される |
| 会議時間選択 | 会議時間ボタン選択 | 主催者が許可した会議時間のみ選択可能。選択に応じてスロット表示を再計算 | スロット一覧が選択した会議時間に合わせて更新 |
| カレンダー連携（任意） | Google/MSログインボタンクリック | OAuthフローで認証。ゲストのカレンダー予定を取得 | カレンダービュー上にゲスト自身の予定をオーバーレイ表示（半透明） |
| 会議モード選択 | オンライン/対面（社内）/対面（訪問）から選択 | 主催者が許可した形式のみ選択可能。形式を切り替えるとバッファー時間が変わるため、空きスロットをAPIで再取得・再表示する | オンライン→カスタム会議URL入力欄を表示。対面（社内）→会議室情報を表示。対面（訪問）→訪問先住所を表示。スロット一覧が更新される |
| 会議ツール指定（任意） | カスタム会議URL入力 | URL形式のバリデーション | 入力されたURLが予約情報に反映。未入力の場合はGoogle Meet自動生成 |
| 予約情報入力 | 名前（必須）、メールアドレス（必須）、メモ（任意） | バリデーション（名前必須、メール形式チェック） | フォームに値が反映される |
| 予約確定 | 予約確定ボタンクリック | ダブルブッキングチェック（主催者側の予定重複確認）。予約データをDB保存。Googleカレンダーにイベント作成（参加者全員を招待）。Google Meet URL自動生成（カスタムURL未指定時）。会議室予約（対面時）。確認メールを全参加者に送信 | 成功→予約完了画面表示。ダブルブッキング検知→エラー「この時間は既に予約されています」 |
| 予約完了表示 | 予約確定成功 | 予約情報を整形して表示 | 予約日時、会議情報（URL/場所）、参加者一覧を表示 |
| エラー表示 | URLのスラッグ | APIでスケジュールリンクを取得し、存在しない・削除済み・無効化済みかを判定 | エラー画面「このリンクは無効です」を表示。予約UIは非表示 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| カレンダービュー | 空き時間スロット表示、日付選択 | ローディング / 表示中（空き日付ハイライト） / エラー（リンク無効・期限切れ） |
| 時間枠リスト | 時間スロット選択 | 非表示（日付未選択） / 表示中（スロット一覧） / 空（選択日に空きなし） |
| 会議時間ボタン | 会議時間選択 | 未選択 / 選択中（ハイライト） |
| Google/MSログインボタン | カレンダー連携 | デフォルト / 連携済み（ボタンが「連携中」表示に変化） |
| オーバーレイ予定 | カレンダー連携 | 非表示（未連携） / 表示中（半透明でゲストの予定表示） |
| 会議モード選択 | 会議モード選択 | オンライン選択中 / 対面（社内）選択中 / 対面（訪問）選択中 / 非表示（単一形式のみ許可の場合）。切り替え時にスロット一覧がローディング→再表示 |
| カスタム会議URL入力 | 会議ツール指定 | 非表示（対面時） / 表示中（オンライン時） / エラー（URL形式不正） |
| 予約フォーム | 予約情報入力 | 非表示（スロット未選択） / 表示中 / バリデーションエラー |
| 予約確定ボタン | 予約確定 | 無効（入力不完全） / デフォルト / ローディング（処理中） |
| 予約完了画面 | 予約完了表示 | 非表示（予約前） / 表示中（予約情報サマリー） |
