# 要件定義書: 予定調整作成画面（P003）

## 目的

スケジュールリンクを新規作成または編集する。参加者の選択、対応可能時間帯の設定、会議オプションの設定を行い、共有用URLを生成する。

## 機能

| # | 機能 | 説明 | v1からの変更点 |
|---|------|------|--------------|
| 1 | タイトル・説明入力 | リンクの名前と説明を入力 | 新規 |
| 1b | カスタムスラッグ設定 | リンクURLの末尾（スラッグ）を任意の文字列に設定・編集 | 新規 |
| 2 | 参加者選択 | 社内メンバー検索・選択、外部メール追加。連携済み外部ユーザーは空き時間計算に含む | 新規 |
| 2b | カレンダー連携招待 | 外部参加者にカレンダー連携招待リンクを送信 | 新規 |
| 3 | 会議時間選択 | 会議の所要時間を選択（15/30/60/90分） | 新規 |
| 4 | 対応可能時間帯設定 | カレンダーグリッドで曜日別の時間帯をドラッグ設定 | 新規 |
| 4b | 祝日除外設定 | 祝日を自動的に調整不可にするON/OFF | 新規 |
| 4c | 日付オーバーライド | 特定日に個別の調整枠を設定（祝日に枠作成、特定日を除外） | 新規 |
| 5 | 空き時間自動計算 | 参加者のカレンダーから空き時間を自動抽出・表示 | 新規 |
| 6 | 会議形式選択 | ゲストに許可する形式を選択（オンライン / 対面・社内 / 対面・訪問、複数選択可） | 新規 |
| 7 | バッファー時間設定 | 形式ごとの前後バッファー時間を設定（オンライン: 0分、訪問: 60分 等） | 新規 |
| 8 | 会議室選択 | 対面（社内）許可時に会議室を選択 | 新規 |
| 9 | 訪問先入力 | 対面（訪問）許可時に先方の場所名・住所を入力 | 新規 |
| 10 | リンク生成・コピー | 保存してURLを生成・コピー | 新規 |
| 11 | 既存設定ロード | 編集モード時に既存設定を読み込み | 新規 |

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| タイトル・説明入力 | タイトル（必須、最大100文字）、説明（任意、最大500文字） | バリデーション（タイトル必須・文字数チェック） | フォームに値が反映される |
| カスタムスラッグ設定 | スラッグ文字列（任意、最大50文字、半角英数字・ハイフンのみ） | 未入力時は自動生成。入力時はユニーク制約チェック（API問い合わせ）。使用可能文字バリデーション | URLプレビュー表示（`boostcal.app/schedule/〇〇〇`）。重複時はエラー「このURLは既に使用されています」 |
| 参加者選択 | 検索キーワード（名前/メール） or 外部メールアドレス | 社内メンバー: APIで組織メンバーを検索。外部: メール形式バリデーション。外部メール追加時にDBでGUEST登録済み（カレンダー連携済み）か確認 | 選択済み参加者リストに追加（連携ステータス表示: 連携済み/未連携）。連携済みユーザーは空き時間計算に含む。未連携ユーザーは空き時間計算対象外 |
| カレンダー連携招待 | 未連携の外部参加者の「連携を依頼」ボタンクリック | カレンダー連携用の招待リンクを生成。招待メールを送信（リンク先でGoogleログイン→カレンダーアクセス許可） | メール送信完了トースト表示。相手が連携完了すると参加者リストのステータスが「連携済み」に更新され、空き時間計算に含まれる |
| 会議時間選択 | 会議時間ボタン選択（15/30/60/90分） | 選択した時間をスケジュールリンク設定に反映 | 選択中の会議時間がハイライト表示 |
| 対応可能時間帯設定 | カレンダーグリッド上でのドラッグ操作 | ドラッグ範囲から時間帯を算出し曜日別デフォルトとして設定に追加。既存ブロックの移動・リサイズ・削除も可能 | カレンダーグリッドに対応可能時間帯ブロック（青色）が表示 |
| 祝日除外設定 | 祝日除外トグルのON/OFF | ON: 祝日（@holiday-jp）に該当する日を調整不可に。OFF: 祝日も通常の曜日設定通りに表示 | カレンダーグリッドで祝日のグレー表示がON/OFF。設定に保存 |
| 日付オーバーライド | カレンダーグリッドで特定日付をクリック→個別スロット設定 | 曜日別デフォルトを上書き。特定日に個別の時間帯を設定（空配列=その日を除外）。祝日除外ONでも個別に枠を作成可能 | 該当日のカレンダーグリッドにオーバーライド枠（青+バッジ表示）が表示 |
| 空き時間自動計算 | 参加者リスト、対応可能時間帯設定 | 自分のGoogleカレンダー予定を取得。社内メンバーはDWD経由でカレンダー取得。連携済み外部ユーザーは保存済みトークンでカレンダー取得。未連携外部ユーザーはスキップ。対応可能時間帯からビジー時間・祝日・過去を除外 | カレンダーグリッドに空き時間（青）とビジー（灰）を表示 |
| 会議形式選択 | オンライン/対面（社内）/対面（訪問）のチェックボックス（複数選択可） | 選択された形式に応じたUI要素を表示。複数選択時はゲスト側で形式を選べるようになる | チェック済みの形式がリンク設定に保存される |
| バッファー時間設定 | 形式ごとのバッファー分数（数値入力） | 各形式の前後バッファーを設定に保存。ゲストが形式を選んだ際にバッファーを加味した空きスロットを計算する | オンライン: デフォルト0分、訪問: デフォルト60分。社内: デフォルト0分 |
| 会議室選択 | 会議室ドロップダウンから選択 | Google Workspaceの会議室リソース一覧を取得・表示 | 選択した会議室が設定に反映。予約時に会議室も予約される |
| 訪問先入力 | 場所名（必須）、住所（任意） | バリデーション（場所名必須チェック） | 入力した訪問先情報が設定に反映。カレンダー招待にも場所として記載される |
| リンク生成・コピー | 保存ボタンクリック | 全設定をバリデーション（タイトル必須・100文字以内、参加者1名以上、会議形式1つ以上選択、対面・訪問チェック時は訪問先場所名必須）。APIでスケジュールリンクを作成/更新。URLスラッグを生成 | スケジュールリンクURL表示。コピーボタンでクリップボードにコピー。ダッシュボードへ遷移可能 |
| 既存設定ロード | URLパラメータのリンクID | APIでスケジュールリンクの詳細設定を取得 | 全フォーム要素に既存設定を反映。ページタイトルが「編集」に変わる |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| タイトル入力フィールド | タイトル・説明入力 | デフォルト / 入力中 / エラー（空のまま保存しようとした時） |
| スラッグ入力フィールド | カスタムスラッグ設定 | デフォルト（プレースホルダー: 自動生成） / 入力中（URLプレビュー更新） / チェック中（ユニーク検証ローディング） / 利用可能（緑チェック） / エラー（重複・不正文字） |
| 参加者検索フィールド | 参加者選択 | デフォルト / 検索中（候補ドロップダウン表示） / 候補なし |
| 選択済み参加者リスト | 参加者選択 | 空（未選択） / 1名以上表示（各参加者に削除ボタン + 連携ステータス: 連携済み/未連携/連携依頼中） |
| 連携依頼ボタン | カレンダー連携招待 | 非表示（社内メンバー・連携済み） / 表示中（未連携外部参加者） / 送信済み（依頼中） |
| 会議時間ボタン | 会議時間選択 | 未選択 / 選択中（ハイライト） |
| カレンダーグリッド | 対応可能時間帯設定、空き時間自動計算 | ローディング / 表示中（空き時間ブロック＋ビジーブロック＋祝日グレー＋オーバーライドバッジ） / 参加者未選択（空き時間計算不可） |
| 祝日除外トグル | 祝日除外設定 | ON（デフォルト、祝日グレー表示） / OFF（祝日も通常表示） |
| 会議形式チェックボックス | 会議形式選択 | 各形式のON/OFF（複数選択可。最低1つ必須） |
| バッファー時間入力 | バッファー時間設定 | チェック済みの形式ごとに表示 / 非表示（未チェック形式） |
| 会議室ドロップダウン | 会議室選択 | 非表示（対面・社内未チェック時） / 表示中 / ローディング / 選択済み |
| 訪問先入力フィールド | 訪問先入力 | 非表示（対面・訪問未チェック時） / 表示中 / エラー（場所名未入力） |
| 保存ボタン | リンク生成・コピー | デフォルト / 無効（バリデーション未通過） / ローディング（保存中） |
| URL表示エリア | リンク生成・コピー | 非表示（保存前） / 表示中（URL + コピーボタン） |
