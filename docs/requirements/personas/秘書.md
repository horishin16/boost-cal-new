# ペルソナ: 秘書

## ユーザー属性

| 項目 | 内容 |
|------|------|
| 名前 | 山田 美咲 |
| 年齢 | 20代 |
| 職業 | 役員秘書 |
| 部門 | 経営企画室 |
| 勤務地 | 本社オフィス |
| 技術スキル | 高め（業務効率化ツールに精通、新しいツールの習得も早い） |

## 目標・ゴール

- 担当役員のスケジュールを正確かつ効率的に管理したい
- 役員に代わって予定調整をスムーズに完了させたい
- 会議室予約・オンライン会議設定・移動時間の考慮を漏れなく行いたい

## 課題

- 役員の予定は変更が多く、最新の状態を常に把握するのが大変
- グループ会社の役員同士の会議調整では、相手側の秘書とのやり取りが何往復にもなる
- 会議室の空き確認、オンライン会議URL発行、移動時間の計算を毎回手動で行っている
- 複数の役員を担当する場合、ダブルブッキング防止に細心の注意が必要

## 利用状況

| 利用場面 | 頻度 | 詳細 |
|---------|------|------|
| 役員の予定代理調整 | 毎日・多数 | 役員に代わり社内外の会議を設定 |
| 会議室・設備予約 | 毎日 | 会議に必要な会議室やオンライン会議の手配 |
| 予定変更・リスケ対応 | 毎日 | 急な予定変更時の再調整と関係者への連絡 |
