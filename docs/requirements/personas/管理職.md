# ペルソナ: 管理職

## ユーザー属性

| 項目 | 内容 |
|------|------|
| 名前 | 佐藤 恵子 |
| 年齢 | 40代 |
| 職業 | 部長 |
| 部門 | 事業企画部 |
| 勤務地 | 本社オフィス |
| 技術スキル | 中程度（基本的なツールは使えるが新しいツールには慣れが必要） |

## 目標・ゴール

- グループ会社横断のプロジェクト会議を効率的に調整し、意思決定スピードを上げたい
- 自分のスケジュールを秘書やアシスタントと共有・代理調整できるようにしたい
- 会議の準備や付随作業を最小化し、マネジメント業務に集中したい

## 課題

- グループ会社をまたぐ会議では参加者が多く、全員の空き時間を合わせるのに非常に時間がかかる
- 秘書に予定調整を依頼しても、自分の最新の予定が正確に共有されていないことがある
- 会議が立て込むとダブルブッキングや予定漏れが発生しやすい
- リマインドがないと重要な会議を忘れるリスクがある

## 利用状況

| 利用場面 | 頻度 | 詳細 |
|---------|------|------|
| グループ横断会議の調整 | 週数回 | 複数会社・部署のメンバーとの会議設定 |
| 部内定例会議の管理 | 週1回 | 定期的なチーム会議の運営 |
| 予定の確認・リマインド | 毎日 | 翌日の予定確認と当日のリマインド受信 |
