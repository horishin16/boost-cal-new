# ペルソナ: IT管理者

## ユーザー属性

| 項目 | 内容 |
|------|------|
| 名前 | 鈴木 大輔 |
| 年齢 | 30代 |
| 職業 | 社内SE / ITインフラ管理者 |
| 部門 | 情報システム部 |
| 勤務地 | 本社オフィス |
| 技術スキル | 高い（システム導入・運用の専門知識あり） |

## 目標・ゴール

- グループ会社横断で安全に利用できるスケジュール管理システムを導入・運用したい
- 各グループ会社の既存カレンダーシステムとの連携を実現したい
- ユーザーからの問い合わせやトラブルを最小限に抑えたい

## 課題

- グループ会社ごとにカレンダーシステムやドメインが異なり、統合が難しい
- セキュリティ要件（アクセス制御、データの取り扱い）を満たしつつ利便性を確保する必要がある
- 導入後の問い合わせ対応やユーザーサポートの負荷が懸念される
- 各社のIT部門との調整・合意形成が必要

| トラブルシューティング | 随時 | 同期エラーや接続問題の対応 |
| 利用状況モニタリング | 月次 | 利用統計の確認とレポート作成 |
