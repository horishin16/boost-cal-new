# 仕様書: 予定調整作成画面

## ページID
P003

## 目的

スケジュールリンクを新規作成または編集する画面。参加者の選択、対応可能時間帯の設定、会議オプションの設定を行い、共有用URLを生成する。新規作成と編集で同じ画面を使用する。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | タイトル・説明入力 | スケジュールリンクの名前と説明を入力する |
| 1b | カスタムスラッグ設定 | リンクURLの末尾（スラッグ）を任意の文字列に設定・編集できる |
| 2 | 参加者選択 | 社内メンバー（グループ会社横断）や外部メールアドレスを参加者として追加する。カレンダー連携済みの外部ユーザーは空き時間計算に含まれる |
| 2b | カレンダー連携招待 | 外部参加者にカレンダー連携を依頼する招待リンクを送信する |
| 3 | 会議時間選択 | 会議の所要時間を選択する（15/30/60/90分） |
| 4 | 対応可能時間帯設定 | カレンダーグリッドで曜日別の対応可能時間帯をドラッグで設定する |
| 4b | 祝日除外設定 | 祝日を自動的に調整不可にするかどうかを切り替える |
| 4c | 日付オーバーライド | 特定の日付に個別の調整枠を設定する（祝日や休日にあえて枠を作成、または特定日を除外） |
| 5 | 空き時間自動計算 | 自分と参加者のカレンダー予定から空き時間を自動抽出・表示する |
| 6 | 会議形式選択 | ゲストに許可する形式を選択（オンライン / 対面（社内） / 対面（訪問）、複数選択可） |
| 7 | 会議室選択 | 対面（社内）を許可する場合、会議室を選択する |
| 7b | 訪問先入力 | 対面（訪問）を許可する場合、先方の住所・場所名を入力する |
| 7c | バッファー時間設定 | 形式ごとの前後バッファー時間を設定する（オンライン: デフォルト0分、訪問: デフォルト60分） |
| 8 | リンク生成・コピー | 設定を保存しスケジュールリンクURLを生成・コピーする |
| 9 | 既存設定ロード | 編集モード時に既存設定を読み込む |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | 戻るボタン、ページタイトル（新規作成 / 編集） | 上部 |
| タイトル入力フィールド | リンク名の入力欄 | フォーム上部 |
| スラッグ入力フィールド | URLの末尾文字列の入力欄（プレビュー付き: `boostcal.app/schedule/〇〇〇`） | タイトル下 |
| 説明入力フィールド | リンクの説明テキストエリア | スラッグ下 |
| 参加者選択エリア | メンバー検索 + 選択済みリスト + 連携ステータス表示 | フォーム中部 |
| 会議時間選択 | 15/30/60/90分のボタングループ | フォーム中部 |
| カレンダーグリッド | 週表示カレンダー（空き時間・ビジー表示、ドラッグで時間帯設定。祝日はグレー表示） | フォーム中央（大） |
| 祝日除外トグル | 祝日を自動除外するかどうかのON/OFFスイッチ | カレンダーグリッド上部 |
| 会議形式チェックボックス | 許可する形式の選択（オンライン / 対面・社内 / 対面・訪問、複数選択可） | フォーム下部 |
| バッファー時間設定 | 形式ごとの前後バッファー分数入力 | 会議形式の下 |
| 会議室選択ドロップダウン | 会議室一覧（対面・社内チェック時のみ表示） | フォーム下部 |
| 訪問先入力フィールド | 場所名・住所の入力欄（対面・訪問チェック時のみ表示） | フォーム下部 |
| 保存ボタン | リンクを保存する | フォーム最下部 |
| 生成されたURL表示 + コピーボタン | 保存後にURLを表示・コピー | 保存ボタン下 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| タイトル・説明入力 | タイトル・説明フィールド | テキスト入力 |
| 参加者選択 | 参加者選択エリア | 名前・メールで検索して選択、または外部メールを直接入力 |
| 会議時間選択 | 会議時間選択ボタン | クリックで選択 |
| 対応可能時間帯設定 | カレンダーグリッド | ドラッグで時間帯を追加・編集 |
| 空き時間自動計算 | カレンダーグリッド | 参加者選択後に自動計算し、空き時間（青）・ビジー（灰）を表示 |
| 会議形式選択 | 会議形式チェックボックス | チェックで形式を許可。複数選択可。チェック状態に応じて会議室・訪問先・バッファーのUI表示を切り替え |
| バッファー時間設定 | バッファー時間設定 | チェック済みの形式ごとにバッファー分数を入力。ゲスト側で形式切替時にスロット計算に反映される |
| 会議室選択 | 会議室選択ドロップダウン | 対面（社内）チェック時にドロップダウンから選択 |
| 訪問先入力 | 訪問先入力フィールド | 対面（訪問）チェック時に先方の場所名・住所を入力 |
| リンク生成・コピー | 保存ボタン → URL表示 → コピーボタン | 保存でURL生成、コピーボタンでクリップボードにコピー |
| 既存設定ロード | 全フォーム要素 | 編集モード時にURLパラメータから既存設定を読み込み反映 |
