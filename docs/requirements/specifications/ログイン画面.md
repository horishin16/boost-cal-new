# 仕様書: ログイン画面

## ページID
P001

## 目的

グループ会社の従業員がGoogle OAuthでログインし、システムにアクセスする。ドメイン制限により許可されたグループ会社の従業員のみがログインできる。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | Googleログイン | Google OAuth 2.0 + PKCEフローでログインする |
| 2 | ドメイン制限 | 許可されたドメイン（boostconsulting.co.jp / boostcapital.co.jp）のみログイン可能 |
| 3 | 自動リダイレクト | ログイン済みの場合はダッシュボードへ自動遷移する |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| アプリロゴ | BoostCalのロゴ | 中央上部 |
| アプリ名・説明 | 「BoostCal - スケジュール調整」 | ロゴ下 |
| Googleでログインボタン | Google OAuthフローを開始するボタン | 中央 |
| ドメイン案内テキスト | 対象ドメインの案内（boostconsulting.co.jp / boostcapital.co.jp） | ボタン下 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| Googleログイン | Googleでログインボタン | クリックでGoogle OAuth画面へ遷移 |
| ドメイン制限 | ドメイン案内テキスト | 許可ドメイン外のアカウントでログイン試行時にエラー表示 |
| 自動リダイレクト | - | ページ読み込み時にセッション確認、ログイン済みならダッシュボードへ遷移 |
