# 仕様書: 調整回答画面

## ページID
P004

## 目的

スケジュールリンクのURLを受け取った相手（ゲスト）が、空き時間スロットを確認して予約を行う画面。認証不要で利用可能だが、Google/Microsoftログイン連携（任意）で自分の予定をオーバーレイ表示できる。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 空き時間スロット表示 | 主催者が設定した空き時間スロットをカレンダービューで表示する |
| 2 | 日付選択 | カレンダーから予約したい日付を選択する |
| 3 | 時間スロット選択 | 選択した日付の利用可能な時間枠から希望の時間を選ぶ |
| 4 | 会議時間選択 | 許可された会議時間（15/30/60/90分）から選択する |
| 5 | Google/MSログイン連携（任意） | ログイン連携で自分のカレンダー予定をオーバーレイ表示する |
| 6 | 会議モード選択 | 主催者が許可した会議形式（オンライン/対面・社内/対面・訪問）から選択する。形式を切り替えるとバッファー時間に応じて空きスロットが動的に更新される |
| 7 | オンライン会議ツール指定（任意） | Zoom/Teams等のカスタム会議URLを入力する |
| 8 | 対面情報表示 | 対面（社内）の場合は会議室情報、対面（訪問）の場合は訪問先住所を表示する |
| 9 | 予約情報入力 | 名前、メールアドレス、メモを入力する |
| 10 | 予約確定 | 予約を確定し、参加者全員にカレンダー招待メールを送信する |
| 11 | ダブルブッキングチェック | 予約確定時に主催者側のダブルブッキングがないか自動チェックする |
| 12 | 予約完了表示 | 予約完了後の確認情報を表示する |
| 13 | エラー表示 | 無効なリンク（削除済み・存在しない・無効化済み）にアクセスした場合にエラーメッセージを表示する |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| リンク情報ヘッダー | 主催者名、リンクタイトル、説明 | 上部 |
| Google/MSログインボタン | カレンダー連携のためのログインボタン（任意） | ヘッダー下 |
| カレンダービュー | 月表示カレンダーで日付を選択 | 左側 |
| 利用可能スロット一覧 | 選択した日付の空き時間枠リスト | 右側 |
| 自分の予定オーバーレイ | ログイン連携時に自分の予定を重ねて表示 | カレンダービュー内 |
| 会議時間選択 | 会議時間のボタングループ | スロット一覧上部 |
| 会議モード選択 | オンライン/対面（社内）/対面（訪問）の選択 | カレンダービュー上部（スロット選択前に表示。切り替えでスロット再計算） |
| カスタム会議URL入力 | Zoom/Teams等のURL入力欄 | オンライン選択時、予約フォーム内（任意） |
| 対面情報表示 | 会議室情報または訪問先住所の表示 | 対面選択時に表示 |
| 予約フォーム | 名前、メールアドレス、メモ入力欄 | スロット選択後に表示 |
| 予約確定ボタン | 予約を確定するボタン | フォーム最下部 |
| 予約完了画面 | 予約日時、会議情報、参加者の確認表示 | 予約確定後に切り替わる |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 空き時間スロット表示 | カレンダービュー、利用可能スロット一覧 | ページ読み込み時にAPIから空き時間を取得し表示 |
| 日付選択 | カレンダービュー | 日付クリックで選択、右側にその日のスロット一覧を表示 |
| 時間スロット選択 | 利用可能スロット一覧 | スロットクリックで選択状態に |
| 会議時間選択 | 会議時間選択ボタングループ | クリックで会議時間を選択 |
| Google/MSログイン連携 | Google/MSログインボタン | クリックでOAuthフロー開始、連携後にカレンダー予定をオーバーレイ表示 |
| 会議モード選択 | 会議モード選択 | クリックで形式を切り替え。バッファー時間に応じて空きスロットをAPIで再取得・再表示 |
| オンライン会議ツール指定 | カスタム会議URL入力 | オンライン選択時に入力欄表示。Zoom/Teams等のURLを貼り付け |
| 対面情報表示 | 対面情報表示 | 対面（社内）→会議室情報を表示。対面（訪問）→訪問先住所を表示 |
| 予約情報入力 | 予約フォーム | 名前・メール・メモをテキスト入力 |
| 予約確定 | 予約確定ボタン | クリックで予約API実行、カレンダー招待メール送信 |
| ダブルブッキングチェック | - | 予約確定時にサーバー側で自動チェック、衝突時はエラー表示 |
| 予約完了表示 | 予約完了画面 | 予約成功後に確認情報を表示 |
