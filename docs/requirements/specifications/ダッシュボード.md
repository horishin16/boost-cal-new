# 仕様書: ダッシュボード

## ページID
P002

## 目的

ログイン後のホーム画面。作成済みのスケジュールリンク一覧を表示し、リンクの作成・編集・削除・URLコピーを行う。

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | スケジュールリンク一覧表示 | 自分が作成したスケジュールリンクの一覧を表示する |
| 2 | リンク新規作成 | 新しいスケジュールリンクの作成画面へ遷移する |
| 3 | リンクURLコピー | スケジュールリンクの共有URLをクリップボードにコピーする |
| 4 | リンク編集 | 既存のスケジュールリンクの編集画面へ遷移する |
| 5 | リンク削除 | スケジュールリンクを削除する（確認ダイアログあり） |
| 6 | Google Calendar連携ステータス | Google Calendarとの連携状態を表示する |
| 7 | ログアウト | セッションを終了しログイン画面へ戻る |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ヘッダー | アプリロゴ、ユーザー名、ログアウトボタン | 上部 |
| Google Calendar連携バナー | 連携ステータス表示（未連携時は警告） | ヘッダー下 |
| 新規作成ボタン | スケジュールリンク作成画面への遷移ボタン | リスト上部 |
| リンクカード一覧 | 各リンクの情報カード（名前、URL、参加者数、作成日） | メインエリア |
| URLコピーボタン | 各カード内のURLコピーアイコンボタン | カード右側 |
| 編集ボタン | 各カード内の編集アイコンボタン | カード右側 |
| 削除ボタン | 各カード内の削除アイコンボタン | カード右側 |
| 削除確認ダイアログ | 削除実行前の確認モーダル | 中央オーバーレイ |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| リンク一覧表示 | リンクカード一覧 | ページ読み込み時にAPIから一覧取得し表示 |
| リンク新規作成 | 新規作成ボタン | クリックで予定調整作成画面（P003）へ遷移 |
| URLコピー | URLコピーボタン | クリックでクリップボードにコピー、トースト表示 |
| リンク編集 | 編集ボタン | クリックで予定調整作成画面（P003）の編集モードへ遷移 |
| リンク削除 | 削除ボタン → 削除確認ダイアログ | クリックで確認ダイアログ表示、確認後に削除実行 |
| Calendar連携ステータス | Google Calendar連携バナー | 未連携時は警告バナー表示、連携済みは正常表示 |
| ログアウト | ログアウトボタン | クリックでセッション終了、ログイン画面へ遷移 |
